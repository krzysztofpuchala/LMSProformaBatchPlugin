LMSProformaBatchPlugin
Proforma Batch Converter
Plugin do wsadowej konwersji faktur proforma na faktury VAT w systemie LMS.

Ten plugin umożliwia zbiorczą (wsadową) konwersję faktur proforma na pełne faktury VAT w systemie LMS (Lan Management System – polskim systemie billingowym dla dostawców usług internetowych i telekomunikacyjnych).

Instalacja

Skopiuj katalog pluginu do directory plugins/ w instalacji LMS, tak aby ścieżka była:
plugins/LMSProformaBatchPlugin/
Plugin powinien zostać automatycznie wykryty i zainicjowany przez system LMS po restarcie lub odświeżeniu.
Sprawdź, czy w menu pojawiły się nowe opcje związane z konwersją wsadową faktur proforma.

Użycie
Po instalacji plugin dodaje nową funkcjonalność w sekcji faktur.
Pozwala na zaznaczenie wielu faktur proforma i ich zbiorczą konwersję na faktury VAT.
